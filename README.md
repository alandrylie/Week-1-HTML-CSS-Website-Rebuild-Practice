# HTML/CSS Website Rebuild Practice
## Week one Challenge Project for ull-Stack Development with AI with AR Bell
### Goal
Today, you are going to rebuild the visual layout of either:
### walmart.com
 or
 ### kohls.com
#### You will use only HTML and CSS.
+ No JavaScript.
+ No functionality required.
+ No cart logic.
+ No search logic.
+ No dropdown behavior.
+ No real checkout.
+ The goal is to make your page look as close as possible to the real website using the HTML/CSS concepts we have covered in class.


#### In this project I have used the following from HTML 5 & CSS 3
+ Div
+ Span
+ ul li
+ button
+ p
+ Semantic HTML
+ Class
+ id
+ display: flex
+ display: block
+ display: grid
+ flex-direction
+ justify-content
+ align-items
+ font-awesome

##### Key takeaways 
+ Dont overthink and waste time trying to get pixel perfect.
+ Plan your time and monitor how long your taking.
+ I spent around double the time it should have taken due to the above.
